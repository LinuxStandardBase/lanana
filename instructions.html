LSB Registry Application Instructions

When applying for a name in LSB namespaces for Init Scripts, Cron Scripts, Package Names or Provider Names, please follow these guidelines or your application may be ignored:

  1. Check LANANA's web pages to see if the name you wish to reserve is still available.
  2. Send an email to lsb @ lanana.org containing the following information:
    • Your Name
    • Your Email Address
    • Name of the Entity (company, organization, etc.) you represent
    • Name of the Application or Project needing a reserved name.
    • Project Website (if there is one).
    • Suggested reserved names and from which namespace (init|cron|package|provider) desired.
    • For the case of reserving init script namespace(s), please include a decription of each init script's purpose. Example: 
      foobar - start/stop the foobard daemon
We will try and respond to your request in a timely fashion.
